How to use Render to deploy the server? 

On terminal, cd to the mcp folder:
(1)git init 
(2)echo ".venv" > .gitignore
(3)git status
(4)render doesn't support uv, so use pip:
uv pip compile pyproject.toml > requirements.txt
(5)echo "python-3.11.11" > runtime.txt
(6)git status #check the runtime
(7)git add .
(8)git commit -m "ready for deployment"
(9)create a new repo on github and push in an existing repo from the cml
git remote add origin
git branch -M main
git push -u origin main
git remote -v #check the push
(10)sign up on render.com; choose "Deploy a web service" and add the repo link
THE ONLY THING to change is at "Start Command":
python research_server.py
Then deploy
(11)check the sse link:
remote-research.onrender/sse
if there is a session id, it means pass
